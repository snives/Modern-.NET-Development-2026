# Modern .NET Development 2026

This repository contains a progressively structured book for experienced C#,
SQL Server, Windows, IIS, and .NET developers doing modern .NET
development in 2026.

The book rebuilds the reader's mental model from familiar enterprise .NET
development toward the current ecosystem: modern .NET, ASP.NET Core, Linux,
containers, cloud hosting, CI/CD, observability, distributed applications,
Kubernetes, AI-enabled development, architecture, and interview preparation.

Chapters are written with progressive disclosure: each topic begins with why it
exists and where it fits before moving into terminology, mechanics, production
tradeoffs, and hands-on practice.
